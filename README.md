# 编译原理实验合集

编译原理课程六次实验的完整实现，包含 C++ 源码、LaTeX 报告、实验截图，以及一个基于浏览器的实验演示平台。

## 目录结构

```
├── compilation-web/     # Web 实验演示平台（实验一到六）
├── 第一次实验/          # DFA 模拟与可视化
├── 第二次实验/          # 词法分析器（双模式：手工 + 自动）
├── 第三次实验/          # LR(0) 项目集规范族生成器
├── 第四次实验/          # SLR(1) 分析表 + flex/bison 选做
├── 第五次实验/          # 语义分析与符号表管理
├── 第六次实验/          # 中间代码生成（四元式）
├── 代码库/              # 实验配套测试源代码（33 个 .src 文件）
├── 大作业二/            # 大作业实验素材
└── *.pptx              # 各次实验指导 PPT
```

## 实验内容

### 实验一：DFA 模拟与可视化
- **文件**: `dfa.cpp`
- 输入 DFA 五元组定义（字母表、状态集、起始状态、接受状态、转移函数）
- 模拟 DFA 运行，验证字符串接受/拒绝
- 枚举长度 ≤ N 的所有接受字符串
- 生成状态转换图

### 实验二：词法分析器
- **文件**: `scanner.cpp`
- 支持两种模式：手工编写 DFA 和 自动生成
- 识别标识符、关键字、整数、浮点数、运算符、界符
- 支持代码库配套文法（9 个关键字 + 完整运算符集）

### 实验三：LR(0) 项目集规范族生成器
- **文件**: `lr0.cpp`
- 输入文法规则，自动构建 LR(0) 项目集规范族
- 生成 GOTO 状态转移图
- 检测移进-归约冲突和归约-归约冲突

### 实验四：SLR(1) 分析表生成器
- **文件**: `slr.cpp`
- 自动增广文法
- 计算 FIRST 集和 FOLLOW 集
- 构建 LR(0) 项目集规范族
- 生成 SLR(1) 分析表（ACTION + GOTO），利用 FOLLOW 集解决冲突
- **选做**: flex/bison 自动化工具实现四则运算计算器（`calc.l` / `calc.y`）

### 实验五：语义分析与符号表管理
- **文件**: `semantic.cpp`
- 词法分析 → 递归下降语法分析 → 构建抽象语法树 (AST)
- 作用域栈符号表管理（变量声明、函数签名）
- 语义检查：重复声明、未声明变量、类型匹配

### 实验六：中间代码生成
- **文件**: `irgen.cpp`
- 生成四元式中间代码 `(op, arg1, arg2, result)`
- 表达式翻译：运算符优先级（`*`/`/` 先于 `+`/`-`）
- 控制流翻译：`if/else`（JZ/JMP/LABEL）、`while`（LABEL/JZ/JMP）
- 函数声明、参数、返回、输入输出

## Web 实验平台

`compilation-web/` 目录下是一个纯前端实验演示平台，涵盖全部六个实验：

| 实验 | 功能 |
|------|------|
| DFA | 五元组输入、状态图可视化、字符串测试、接受串枚举 |
| LL(1) | 文法输入、FIRST/FOLLOW 计算、预测分析表、推导动画 |
| LR(0) | 文法输入、项目集规范族、状态转移图、冲突检测 |
| SLR(1) | 文法输入、FIRST/FOLLOW 集、ACTION/GOTO 分析表 |
| 语义分析 | 源码输入、AST 树构建、作用域符号表、语义检查 |
| 中间代码 | 源码输入、四元式生成、符号表、Token 流 |

### 启动方式

```bash
cd compilation-web
python -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 技术栈

- **C++ 源码**: GCC/MinGW 编译，Windows 控制台 UTF-8 输出
- **LaTeX 报告**: XeLaTeX + ctexart 中文排版
- **Web 平台**: 原生 HTML/CSS/JS + Tailwind CSS + vis.js 图可视化
